How to Use PSP & True Opens to Identify Your Bias I Kenpachi Trades

A mini guide on how to predict bias using Quarterly Theory concepts. You can find a more detailed explanation on YouTube @kenpachi_qt.
Using PSP — Precision Swing Point

As you all know, we mainly use PSP to confirm that an SSMT has formed. But it has another very powerful use as well.

90Mc SSMT w/ES + 5.15M PSP = 90Mc SSMT confirmed by the 5.15M PSP. We expect that our PSP won't be taken.

PSP is especially powerful for pre-trading analysis: open your charts and work top-down — HTF to MTF. Start on 4H, marking up imbalances or True Opens. Once everything is marked up, jump to D, W, M (monthly — very rarely) and check if any PSP has formed there. If you see an HTF PSP — let's say a Bullish PSP on Daily — your bias switches to longs straight away, because you don't expect price to go below that Daily PSP low anymore. That D PSP low should not get taken. → Bias.

Using True Opens — T.O

True Opens are one of the most important and accessible concepts in Quarterly Theory. We use them as targets, and as premium/discount zone markers.

As targets — if price is bullish above a True Open, we expect it to push towards higher targets (like previous highs or the next PD array). The True Open itself becomes a key level that should not get violated easily in the opposite direction.

Premium / Discount zones — True Opens define where price is "cheap" (below = discount, good for buys if bias is up) or "expensive" (above = premium, good for sells if bias is down). For example: if daily bias is bullish based on HTF, buy dips below the Daily True Open (Midnight NY) treating it as discount — or fade rallies above it if it's acting as premium resistance. Same logic applies to HTF True Opens: TQO, TMO, TWO. Higher timeframe = more probability. → Bias.
